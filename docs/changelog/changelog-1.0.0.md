# Changelog - v1.0.0

**发布日期**: 2026-02-20

IT Toolbox v1.0.0 是首个正式发布版本，包含 15 个核心工具，覆盖开发者日常最高频的使用场景。

---

## 新增功能

### 格式化工具

#### JSON 格式化
- JSON 数据格式化，支持 2/4 空格缩进
- JSON 数据压缩（去除空白）
- JSON 语法校验，显示错误位置
- 支持大文件处理

#### Markdown 预览
- 实时渲染 Markdown 内容
- 支持 GFM（GitHub Flavored Markdown）
- 支持数学公式渲染
- 代码语法高亮
- 导出 HTML 功能

### 编码解码工具

#### Base64 编解码
- 文本 Base64 编码/解码
- Base64URL 编码/解码
- 文件转 Base64 功能
- 支持拖拽上传文件

#### URL 编解码
- encodeURIComponent 编码/解码
- encodeURI 完整 URL 编码/解码
- URL 参数解析和展示

#### JWT 解析
- 解析 JWT Header 和 Payload
- 时间戳自动转换为可读时间
- 过期时间高亮提示
- 支持多种 JWT 算法识别

### 加密哈希工具

#### Hash 计算器
- 支持 MD5、SHA-1、SHA-256、SHA-384、SHA-512
- 文本哈希计算
- 文件哈希计算（支持拖拽上传）
- 一键复制哈希值

#### 密码生成器
- 可配置密码长度（8-64 位）
- 可选字符集：大写字母、小写字母、数字、特殊符号
- 排除相似字符选项（0/O, 1/l/I）
- 密码强度可视化展示
- 破解时间估算
- 批量生成（最多 100 个）

### 数据生成工具

#### UUID 生成器
- UUID v4 批量生成（最多 1000 个）
- 大写/小写格式选项
- 带连字符/无连字符选项
- 一键复制或批量导出

### 时间日期工具

#### 时间戳转换
- Unix 时间戳转日期时间
- 日期时间转 Unix 时间戳
- 支持秒/毫秒单位
- 全球时区支持
- 自动识别当前时间

### 文本处理工具

#### 命名规范转换
- camelCase（驼峰命名）
- PascalCase（帕斯卡命名）
- snake_case（蛇形命名）
- kebab-case（短横线命名）
- CONSTANT_CASE（常量命名）
- 大写/小写转换
- 标题格式/句子格式

#### 占位文本生成
- 英文 Lorem Ipsum 生成
- 中文乱数假文生成
- 可配置段落数
- 可配置单词数

#### 正则表达式测试
- 实时匹配高亮
- 分组提取展示
- 多行模式支持
- 常用正则表达式库
- 支持多种标志（g、i、m 等）

#### 文本统计
- 字符数统计（含/不含空格）
- 单词数统计
- 行数统计
- 字节数统计
- 段落数统计
- 句子数统计
- 阅读时间估算
- 朗读时间估算

### 颜色设计工具

#### 颜色选择器
- HEX/RGB/HSL/HSV/CMYK 格式互转
- 可视化颜色选择器
- 颜色历史记录
- 一键复制各种格式

### 转换工具

#### 进制转换
- 二进制/八进制/十进制/十六进制互转
- 实时转换显示
- 支持大数处理

### 网络工具

#### IP 地址查询
- 自动获取当前 IP 信息
- 显示城市、国家、地区
- 显示时区信息
- 显示 ASN 和组织信息
- 显示经纬度坐标
- KV 缓存加速（1 小时）

#### DNS 查询
- 支持 A、AAAA、MX、TXT、CNAME、NS、SOA 记录
- 实时查询 Cloudflare 1.1.1.1 DoH
- KV 缓存加速（5 分钟）

### AI 增强工具

#### AI 代码解释
- 支持多种编程语言
- 中文解释代码逻辑
- 最大输入 4000 字符

#### AI 生成正则
- 自然语言描述生成正则表达式
- 返回正则模式和解释
- 自动推荐标志位

#### AI 生成 SQL
- 自然语言描述生成 SQL
- 支持传入 DDL Schema 作为上下文
- 返回 SQL 语句和解释

## 核心功能

### 全局搜索
- `Ctrl/Cmd + K` 快捷键打开搜索面板
- Fuse.js 模糊搜索算法
- 支持中英文关键词搜索
- 支持标签搜索

### 收藏功能
- 收藏常用工具
- 收藏列表持久化存储
- 快速访问收藏工具

### 使用历史
- 自动记录工具使用历史
- 保存最近 10 条输入记录
- 本地持久化存储

### 主题切换
- 亮色主题
- 深色主题
- 跟随系统主题
- 平滑过渡动画

### 响应式设计
- 完美适配桌面端
- 完美适配平板端
- 完美适配移动端

## 技术实现

### 前端技术
- React 18.3 + TypeScript 5.5
- TanStack Router 1.58（类型安全路由）
- Zustand 5.0（状态管理）
- Tailwind CSS 3.4（样式）
- Vite 6.0（构建工具）

### 后端技术
- Hono 4.6（Web 框架）
- Cloudflare Pages Functions（边缘计算）
- Cloudflare KV（缓存）
- Cloudflare R2（文件存储）
- Workers AI（AI 推理）

### 性能优化
- 按路由代码分割
- 静态资源 CDN 缓存
- API 响应 KV 缓存
- 首屏加载 < 1.2s

## 兼容性说明

### 浏览器支持
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### API 兼容性
- 所有 API 接口版本为 v1
- 响应格式保持稳定
- 后续版本将保持向后兼容

- 收藏数据存储在 localStorage

## 已知问题

1. **Safari 14** 下某些 CSS 动画可能不流畅
2. **移动端** 长时间使用可能导致内存占用增加
3. **AI 功能** 复杂请求可能需要 3-5 秒响应时间
