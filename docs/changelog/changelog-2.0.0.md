# Changelog - v2.0.0

**发布日期**: 2026-02-21

IT Toolbox v2.0.0 是一个重大版本更新，从 v1.0.0 的 15 个核心工具扩展到 72 款工具，覆盖开发者的更多使用场景。

---

## 新增功能

### 格式化工具（新增 7 款）

#### YAML ↔ JSON
- YAML 与 JSON 格式互转
- 支持复杂嵌套结构
- 错误提示友好

#### SQL 格式化
- 格式化和美化 SQL 语句
- 支持多种数据库方言（MySQL、PostgreSQL、SQLite 等）
- 支持压缩功能

#### XML 格式化
- 格式化、压缩、校验 XML 文档
- 语法高亮显示
- 错误位置提示

#### CSS 格式化
- CSS/SCSS/Less 格式化与压缩
- 支持多种缩进风格
- 语法校验

#### JS/TS 格式化
- JavaScript/TypeScript/JSON5 格式化与压缩
- 支持多种缩进风格
- 语法校验

#### TOML ↔ JSON
- TOML 与 JSON 格式互转
- 支持 TOML 1.0 规范

#### CSV 查看器
- CSV 数据可视化表格展示
- 支持自定义分隔符
- 支持导出 JSON

### 编码解码工具（新增 8 款）

#### HTML 实体编解码
- HTML 实体编码与解码
- 支持命名实体和数字实体
- 批量处理

#### Hex 编解码
- 文本与十六进制互转
- 支持大小写选择
- 支持空格分隔

#### Unicode 转换
- 字符与 Unicode 码点互转
- 支持多种表示格式（\uXXXX、U+XXXX）
- 批量转换

#### 摩斯密码
- 文本与摩斯电码互转
- 支持字母、数字、标点符号
- 可调节字符间隔

#### 二进制文本
- 文本与二进制字符串互转
- 支持 8 位和可变长度
- 空格分隔可选

#### ROT13/Caesar
- ROT13 及 Caesar 移位密码
- 偏移量可调（1-25）
- 支持大小写保留

#### Punycode 转换
- 国际化域名 IDN Punycode 编解码
- 支持中文域名转换
- 符合 RFC 3492 规范

#### ASCII 码表
- 完整 ASCII/Unicode 字符查询
- 支持描述搜索
- 可点击复制

### 加密安全工具（新增 7 款）

#### AES 加密解密
- AES-GCM 对称加密与解密
- 支持 128/192/256 位密钥
- 自动生成 IV

#### RSA 密钥生成
- 生成 RSA 密钥对
- 支持多种密钥长度（2048/4096 位）
- 导出 PEM 格式

#### HMAC 计算
- HMAC-SHA256/512 签名计算
- 支持自定义密钥
- 结果格式可选

#### Bcrypt 哈希
- Bcrypt 密码哈希与验证
- rounds 可配置（4-12）
- 自动生成 salt

#### SSH 密钥生成
- 生成 Ed25519/RSA SSH 密钥对
- 支持设置密码保护
- 一键复制公钥/私钥

#### JWT 生成
- 填写 Payload/Header，选择算法生成 Token
- 支持 HS256/RS256 算法
- 实时预览

#### TLS 证书解析
- 解析 PEM 证书
- 显示域名/有效期/指纹/SAN
- 支持证书链解析

### 网络 HTTP 工具（新增 7 款）

#### URL 解析
- 解析 URL 的各个组成部分
- 显示协议、主机、路径、查询参数等
- 查询参数表格化展示

#### HTTP 状态码
- 查询 HTTP 状态码的含义和说明
- 分类展示（1xx-5xx）
- 快速搜索

#### HTTP 请求头
- 常见请求/响应头说明
- 安全头建议
- 最佳实践提示

#### MIME 类型
- 文件扩展名与 MIME 类型对照查询
- 支持双向搜索
- 常用类型快速复制

#### IP 子网计算
- CIDR 计算网络地址/广播地址/主机数
- 支持IPv4
- 可视化展示

#### UA 解析
- 解析 User-Agent 字符串
- 识别浏览器/OS/设备
- 显示详细信息

#### cURL 转换
- cURL 命令转换为 fetch/axios/Python/Go 代码
- 支持复杂请求
- 一键复制

### 文本处理工具（新增 7 款）

#### 文本 Diff
- 对比两段文本的差异
- 高亮显示增删改
- 支持行级和字符级对比

#### 文本变换
- 去重/排序/反转/去空行等批量操作
- 支持多种排序方式
- 实时预览

#### 字符串转义
- JS/Python/Java/C 字符串转义与反转义
- 支持多种转义规则
- 批量处理

#### Cron 表达式解析
- 解析和生成 Cron 表达式
- 显示下次执行时间
- 支持标准 Cron 语法

#### 文本相似度
- 计算两段文本的相似度
- 基于 Levenshtein 距离
- 显示编辑距离

#### 行处理工具
- 排序/去重/随机打乱/提取第N行/行号添加
- 支持正则过滤
- 批量操作

#### JSON 转类型定义
- JSON 一键生成 TypeScript/Go/Python/Kotlin 类型
- 支持嵌套结构
- 可配置命名风格

### 颜色设计工具（新增 6 款）

#### CSS 渐变生成器
- 可视化生成 CSS 渐变代码
- 支持线性渐变和径向渐变
- 多色渐变支持

#### 配色方案生成
- 输入主色生成类比/互补/三角配色方案
- 支持多种配色规则
- 一键复制颜色值

#### WCAG 对比度
- 前景/背景色对比度检测
- AA/AAA 级别判定
- 可访问性建议

#### 色盲模拟
- 模拟 8 种色觉缺陷显示效果
- 支持实时预览
- 辅助设计无障碍界面

#### Tailwind 色板
- 完整 Tailwind v3/v4 色板
- 点击复制
- 支持多种格式

#### Box Shadow 生成
- 可视化调整 box-shadow
- 支持多层阴影
- 实时预览效果

### 时间日期工具（新增 4 款）

#### 日期计算器
- 日期加减/两日期差值
- 工作日计算
- 支持多种日期格式

#### 时区转换
- 全球时区对照
- 多时区同屏显示
- 支持自定义时区

#### 时长格式化
- 秒数与 hh:mm:ss/人话描述互转
- 支持多种输出格式
- 本地化显示

#### 日历生成
- 生成指定年月日历
- 支持导出 iCal
- 可自定义起始日

### 生成器工具（新增 6 款）

#### NanoID 生成器
- 自定义字符集和长度
- 批量生成
- 一键复制

#### ULID 生成器
- 时间有序唯一 ID
- 比 UUID 更友好
- 支持解析

#### MongoDB ObjectID
- 生成/解析 ObjectID
- 显示时间戳和机器码
- 批量生成

#### 假数据生成
- 姓名/邮箱/手机/地址/公司等假数据生成
- 支持中文
- 多种数据类型

#### 二维码生成
- 生成二维码
- 支持文本、URL、WiFi 等格式
- 可调整大小和颜色

#### 占位图生成
- 生成指定尺寸/颜色/文字的占位图
- 支持 SVG/PNG 格式
- 可自定义样式

## 改进优化

### IP 查询功能增强
- 支持查询任意 IP 地址（通过 `?ip=` 参数）
- 使用 ip-api.com 免费服务查询任意 IP
- 当前请求 IP 仍使用 Cloudflare 原生数据

### 历史记录功能增强
- 历史记录上限从 10 条增加到 20 条
- 优化历史记录存储结构

### UI/UX 改进
- 优化工具页面布局
- 改进移动端适配
- 优化深色模式配色

### 性能优化
- 优化代码分割策略
- 减少首屏加载体积
- 改进懒加载机制

## 兼容性说明

### 浏览器支持
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### API 兼容性
- 所有 API 接口保持向后兼容
- IP 查询接口新增可选参数 `ip`

- 收藏数据存储在 localStorage（格式不变）

## 已知问题

1. **Safari 14** 下某些 CSS 动画可能不流畅
2. **移动端** 长时间使用可能导致内存占用增加
3. **AI 功能** 复杂请求可能需要 3-5 秒响应时间
4. **RSA 密钥生成** 在低端设备上可能较慢
